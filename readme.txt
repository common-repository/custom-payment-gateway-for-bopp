== Description ==
Extends WooCommerce to process payments with BOPP gateway. BOPP WooCommerce plugin extends the functionality of WooCommerce by adding a BOPP pay option to the checkout page. BOPP is an Open Banking payments solution that allows immediate confirmation of bank  transfers. Open Banking payments trough BOPP network are processed by Faster Payments provider, which allows payments to be processed in real time. 


== Installation ==

= Automatic Installation =
> 1. Sign in to your WordPress site as an administrator.
> 2. In the main menu go to Plugins -> Add New.
> 3. Search for BOPP OB GATEWAY WP and click install.
> 4. Activate the plugin by clicking the Activate link.
> 5. On the WooCommerce Settings -> Payments tab enable BOPP OB GATEWAY WP plugin, and go to the settings link to enter the Merchant Key you received from your BOPP Dashboard.
> 6. Please use Sandbox Merchant Key for testing purpose and Live Merchant Key for live transactions.
> 7. That's it. Now you should see the BOPP OB GATEWAY WP payment option on the checkout page.
> 8. Once the payment is successful, the order status will be updated to Processing.
